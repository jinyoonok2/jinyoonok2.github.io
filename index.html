Jinyoon Kim

I completed my bachelor of science in Computer Science at Pennsylvania State University Harrisburg in 2024. My research interests include enhancing and developing 3D object detection in machine perception and vision language models to help models fluently understand the contents of the world through computer vision, enabling interaction with humans or the environment to perform tasks. Currently, I am working on deep learning neural networks that help autonomous agents safely navigate through space by providing 3D visualizations of their surroundings, generated from real-time video or image input. Before graduation, I worked on various research projects related to image recognition for medical image analysis, as well as automated self-supervised learning systems, and published the results under the supervision of my mentors.

Projects

  • Capstone Project: Skin Cancer Detection Web Application (2024)
    Jinyoon Kim, Tianjie Chen, Hien Nguyen, and Md Faisal Kabir [ slides | project code | web application code ]
    • To create a web application for skin cancer detection, making it accessible and user-friendly
    • Developed the application using YOLOv8, combined ISIC datasets, and implemented confounding factors removal and interpretability techniques
    • Successfully created a PWA-based web application, ensuring wide accessibility and transparent diagnostic processes through visual interpretability
  • Machine Learning Project: Face Recognition Program (2023)
